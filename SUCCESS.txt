🎊 BREAKTHROUGH: QR CODES WORK!
──────────────────────────────────────────────────────────────────
After testing 7 formats, Format 6 confirmed working on iOS:

❌ What DIDN'T work:
  • vpn://base64data (prefix breaks scanner)
  • Plain text
  • Uncompressed JSON
  • Standard Base64 with padding

✅ What WORKS (Format 6):
  • Compact JSON structure
  • Qt compression (4-byte size header + zlib)
  • URL-safe Base64 (no padding)
  • NO vpn:// prefix!

The iOS scanner reads the QR as text, then:
1. Base64-decodes it → compressed bytes
2. qUncompress → JSON
3. Parse → Success!

💡 KEY INSIGHT
──────────────────────────────────────────────────────────────────
The vpn:// prefix that works on desktop BREAKS on iOS!

iOS QR scanner returns the EXACT text from QR code.
Desktop apps strip the prefix, iOS doesn't.
Solution: Generate Base64 data WITHOUT prefix.

This was discovered after testing 7 different formats and
analyzing the iOS-specific code path in importController.cpp.
