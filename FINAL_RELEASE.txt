🔑 THE QR CODE BREAKTHROUGH
──────────────────────────────────────────────────────────────────
After testing 7 different formats, discovered:

❌ Formats that FAILED:
  • Plain text config
  • JSON uncompressed
  • vpn://base64... (with prefix)
  • Full JSON with all fields

✅ Format 6 that WORKS:
  • Compact JSON structure
  • Qt compression (4-byte header + zlib)
  • URL-safe Base64 encoding
  • NO vpn:// prefix! (This was the key!)

Technical details:
  - iOS scanner reads QR text as-is
  - App Base64-decodes first (line 634 in importController.cpp)
  - Then qUncompress
  - Then parse JSON
  - The vpn:// prefix was being included literally, breaking decode!
